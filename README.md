# Number_plate_detection_yolov8
This repository demonstrates Automatic Number Plate Detection using YOLOv8 (Ultralytics).
The workflow includes:

📥 Downloading and preparing the custom Number Plate Dataset
🏋️ Training YOLOv8 on this dataset
🔍 Running inference on test images
🖼️ Visualizing bounding box predictions
This project is useful for building ANPR (Automatic Number Plate Recognition) systems in real-world applications like traffic monitoring, toll booths, and parking management.
